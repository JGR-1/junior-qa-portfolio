# Unintended tab switching while scrolling between "Live Channels" and "Categories"

Test Case ID: TC-002

Precondition:
- User is logged into the Twitch mobile application

- User is on the Search screen

Steps
1. Tap on the Search icon
2. Search for any tag or key word
3. Select "Live Channels" tab
4. Scroll down at a decent speed standar speed for an User


Expected Result:
- The application should remain in the Live Channels tab

- Tab switching should require a clear horizontal swipe or intentional gesture

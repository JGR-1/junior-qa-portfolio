Application handles rapid consecutive order submissions without crashing

Precondition:
- User is on the order page
- Page is fully loaded

Steps:
1. Click any item on the menu
2. Select any extra option
3. Submit the order requested
4. Quickly modify the selected option
5. Submit the order request again multiple times without waiting for confirmation

Expected result:
- Application should process user inputs correctly
- No crashes, freezes, or unexpected behavior should occur
- Selected options should match the user's last input

Source: https://www.mcdonalds.com.ar/

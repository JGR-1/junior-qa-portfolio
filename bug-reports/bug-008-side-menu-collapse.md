Side navigation menu collapses when navigating between pages

Bug type: UX

Severity: Low

Priority: Medium

Steps to reproduce:
1. Be on any section of the page
2. Select any option on the side menu

Expected result:
- When the option is selected and the page is loaded, the folder options on the menu should stay open as per user selection.

Actual result:
- The menu collapses every time the page is reloaded, either from an option is selected or just refresh, making the user open the menu again everytime.

Enviorment:
- Chrome 120
- Windows 10

Source: https://www.d20pfsrd.com/
